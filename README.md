#### CSS Resets
A CSS reset removes (or “resets”) default browser styling (margins, paddings, headings, etc.) to a blank baseline.
Different browsers apply different defaults, so resets aim for consistency across them.
They often set everything to zero or neutral values, giving you full control over styling.
However, they can be aggressive and remove useful defaults like heading sizes or list styles.
You then need to rebuild styles from scratch for all elements.

#### Normalize.css
Normalize.css is a more modern alternative that preserves useful browser defaults instead of removing everything.
It fixes inconsistencies between browsers while keeping sensible styling intact.
For example, it standardizes things like line heights, font sizes, and form element behavior.
It’s less destructive than a reset, so you don’t have to re-style everything manually.
Overall, it provides a consistent, “normalized” foundation rather than a blank slate.

### Rule of Thirds
The rule of thirds divides a webpage into a 3×3 grid, and places key elements along the grid lines or at their intersections to create a naturally balanced, visually engaging layout. It guides where to position headlines, images, and calls-to-action so the design feels intentional rather than centered and static.
