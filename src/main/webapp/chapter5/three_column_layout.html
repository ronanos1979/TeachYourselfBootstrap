Three Column Layout

Demonstrates how Bootstrap's col-md-* classes create multi-column layouts. Columns stack vertically on small screens (below md, i.e. <992px) and sit side by side on medium screens and above.

Some default text should never display;

Example 1 — Three equal columns (each col-md-4 = 4/12 = one third)

Resize below 992px: all three columns stack into a single full-width column.

Example 3 — Responsive columns: different widths at each breakpoint

Col 1: col-xs-12 col-sm-6 col-md-4 col-lg-2 — gets narrower as the screen grows.
Col 2: col-xs-12 col-sm-6 col-md-4 — no lg rule, so it keeps md width on large screens.
Col 3: col-xs-12 col-sm-6 — no md/lg rule, so it keeps sm width on larger screens.
