Responsive Web Layout

These columns use multiple breakpoint classes so their widths change at each screen size. Resize your browser slowly from wide to narrow to watch the layout adapt.

Breakpoints in Bootstrap 3: xs <768px | sm ≥768px | md ≥992px | lg ≥1200px

Some default text should never display;

Three columns: col-sm-6 col-md-4 col-lg-3 col-lg-offset-1

sm (≥768px): each column is col-sm-6 (half width) — only 2 fit per row, so col 3 wraps.
md (≥992px): each column is col-md-4 (one third) — all 3 fit side by side.
lg (≥1200px): each column is col-lg-3 (one quarter) with col-lg-offset-1 (one column gap to the left) — the offset adds visual separation but also means 3×(3+1)=12, perfectly filling the row.

Column 1
sm: half-width (col-sm-6)
md: one third (col-md-4)
lg: one quarter + 1-col offset (col-lg-3 col-lg-offset-1)
Column 2
sm: half-width (col-sm-6)
md: one third (col-md-4)
lg: one quarter + 1-col offset (col-lg-3 col-lg-offset-1)
Column 3
sm: wraps to next line (2 × col-sm-6 already fills the row)
md: one third (col-md-4)
lg: one quarter + 1-col offset (col-lg-3 col-lg-offset-1)
